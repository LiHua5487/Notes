
# 1

自动机是一种数学模型，通过读取输入序列，根据当前状态和输入符号按照设定好的迁移函数转移到下一个状态，并判断有没有达到目标状态

联系：KMP 可以看作自动机的一种特例，即专门为了字符串匹配而设计的自动机，其中的 next 数组就相当于状态转移函数，当匹配失败时将模式串移动到相应的位置，这与自动机的思路一致

区别：自动机规定了所有情况下的状态转移，而 KMP 的 next 数组只是在字符匹配失败情况下的转移方式

---

对于字符串 `ababaca` ，其 next 数组为 `[-1,0,0,1,2,3,0]` ，表示当前位置的字符与目标串不匹配时，应把模式串的哪一位移过来进行比较

而 DFA 构建一个表格，其状态表示前面多少个字符相匹配，状态 7 则表示完全匹配，而这个表格根据当前状态与输入的下一个字符来决定转移到哪个状态

|状态|输入 'a'|输入 'b'|输入 'c'|
|---|---|---|---|
|0|1|0|0|
|1|1|2|0|
|2|3|0|0|
|3|1|4|0|
|4|5|0|0|
|5|1|4|6|
|6|1|0|0|

# 2

声明两个左右指针 `i` 和 `j` ，和一个哈希表 `charIdx` 记录两个指针之间出现的字符（作为键）及其下标（作为值），并用 `maxLeng` 与 `startIdx` 记录最大长度与起始位置

开始时，左指针处于下标 0 ，右指针从左往右滑动，每移动到一个位置，储存这个位置的字符与其下标到 `charIdx` ，直到访问到某个字符，在  `charIdx` 中已经有了这个键，且其值处于左右指针之间，说明发生重复，此时将左右指针的距离与 `maxLeng` 进行比较与更新，而后把左指针移动到 `charIdx` 中这个字符对应的值的下一个位置，把 `charIdx` 中这个字符对应的值更新为右指针处，而后右指针继续移动

```python
charIdx = {}
maxLen, startIdx = 0, 0
i = 0
for j in range(n):
    if str[j] in charIdx and charIdx[str[j]] >= i:
        if j - i > maxLen:
            maxLen = j - i
            startIdx = i
        i = charIdx[str[j]] + 1
    charIdx[str[j]] = j
if n - i > maxLen:
    maxLen = n - i
    startIdx = i	
```

# 3

如果要删除 b 和 ac 并移动剩余字符，只需确定要把保留的字符替换到哪里，为此声明一个指针 w ，用另一个指针 i 遍历 S ，如果是 b 就跳过；如果是 c 且 w 处的前一个字符是 a ，则将 w 回退一位；剩余情况则将 `S[w]` 替换为 `S[i]` ，并让 w 向下移动，遍历完后的 w 就是删除后的长度

```python
w = 0
for i in range(n):
	if S[i] == 'b':
		continue
	elif S[i] == 'c' and w > 0 and S[w-1] == 'a':
		w -= 1
	else:
		S[w] = S[i]
		w += 1
return w
```

由于只进行一次无回退的遍历，时间复杂度是 $O(n)$ 

# 4

只需在 KMP 的每次检测时记录并更新 `first_pos` 即可，当检测到 T 的下标 t 处和 P 的下标 p 处不匹配时，说明 P 中 0~p-1 与 T 匹配，此时可以设置 `first_pos` 的 1~p 项，通过 p 与 t 计算出匹配起始点

在上述更新中，实际上只需设置最大匹配长度 p 的 `first_pos` ，因为更短的长度已经在前面设置过了，为此引入一个变量 `maxLeng` 辅助标记

尽管 KMP 的 next 数组会跳过一些区间，但这些区间中发生的匹配，其长度都是小于 p 的，而且发生在更后边，所以不影响结果

```cpp
# 计算next数组
i, k = 0, -1
next = [-1, ..., -1]
while i < m - 1:
    while k >= 0 and P[i] != P[k]:
        k = next[k]
    i += 1
    k += 1
    next[i] = k

# KMP并更新first_pos
p, t = 0, 0
maxLeng = 0
first_pos = [-1, ..., -1]
while p < m and t < n:
	if p == -1 or T[t] == P[p]:
		p += 1
		t += 1
	else:
		if p > maxLeng:
			for i in range(maxLeng, p):
				first_pos[i + 1] = t - p
			maxLeng = p
		p = next[p]

# 处理末端匹配情况
if p == m:
    if p > maxLeng:
        for i in range(maxLeng, p):
            first_pos[i + 1] = t - p
        maxLeng = p
```
