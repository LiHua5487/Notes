如果有相同的子表出现，那么必然有相同的原子，所以只需递归检查有没有重复原子即可

```
def check(node, visited):
	if node == NULL:
		return true # 是纯表
	
	if node.tag == LIST:
		if node.hp != NULL:
			if not check(node.hp, visited):
				return false
		if node.tp != NULL:
			if not check(node.tp, visited):
				return false
		return true
	else if node.tag == ATOM:
		if visited.contains(node.atom) then
			return false
		else
			visited.add(node.atom)
			return true
	return true
```

空间开销
- Trie 树：每个字符对应一个节点，会有大量节点，如果字符集比较大，指针数组会占用很多内存
- MPT：将单一路径的节点进行压缩，并使用哈希引用，在稀疏数据中空间效率更高

降低开销的方法：使用双数组 Trie ，用数组存储节点和转移，减少指针开销
