# 3

用一个数组 $S$ 存储所有子集和
- 初始时 $S$ 中只有一个元素 0 ，代表空集的和
- 遍历 $A$ 中元素 $x_i$ ，对于 $S$ 中每个元素 $s_j$ ，把 $s_j+x_i$ 加入到 $S$ 中
即可得到所有子集和

在每次添加新元素时，如果 $S$ 本身已经是有序的，那新添加的元素组成的序列 $s_0+x_i,\cdots, s_m+x_i$ 也是有序的，可以用归并操作把它们合并成新的有序序列

```
S = [0]
for x in A:
	tmp = []
	for s in S:
		tmp.append(s+x)
	merge(S,tmp)
```

第 $i$ 次遍历时，$S$ 的规模 $m=2^{i-1}$ ，而归并耗时 $2m$ ，故总时间复杂度为 $\sum_i 2^{i-1} + 2\cdot 2^{i-1} \sim O(2^n)$
