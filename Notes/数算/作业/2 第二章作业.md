
# 1

假设一个节点前驱指针为 `prev` ，后继指针为 `next` ，伪代码如下

```cpp
// 确保节点非空
if(p == Null) return;

p->prev->next = p->next;
p->next->prev = p->prev;
p->prev = Null;
p->next = Null
delete(p);
```

# 2

设数组为 `a` ，长度为 `n` ，使用双指针的方式，声明两个变量 `prev` 和 `curr` ，表示上一个元素与当前访问的元素的下标，伪代码如下

```cpp
if(n <= 1) return;

prev = 0;
for(int curr = 1; curr < n; curr++) {
	// curr向右移动，找到了首个和prev不一样的元素
	if(a[curr] != a[prev]) {
		// prev向右移动，把这个不一样的元素放到下一个位置
		prev++;
		a[prev] = a[curr];
	}
}

return prev + 1;
```

最后 `prev + 1` 即为删除后的长度

# 3

采用快慢指针的方式，声明两个指针 `slow` 和 `fast` ，沿着链表进行移动，慢指针每次移动 1 步，快指针每次移动 2 步，如果二者在某个时刻相遇，说明有环，此时可以推导出入口点

假设入口点到 `head` 的距离为 $d$ ，相遇点到入口的距离为 $r$ ，一圈长度为 $R$ ，相遇前快慢指针在环里分别走了 $m$ 和 $n$ 圈，其中 $m>n$ ，由于快指针速度为慢指针的 2 倍，所以移动距离也为 2 倍，可得以下方程

$$
\begin{aligned}
&d+mR+r=2(d+nR+r) \\
\iff &d=(m-2n)R-r
\end{aligned}
$$

而 $(m-2n)R-r$ 的含义是从相遇点再往下移动到入口处的距离，加上整数倍的圈数，如果我们让这两个指针一个从 `head` 出发，一个从相遇点触发，每次移动 1 步，最后就会在入口点相遇，可得伪代码如下

```cpp
slow = head; fast = head;
while(fast != Null && fast->next != Null) {
	slow = slow->next; // slow走一步
	fast = fast->next->next; // fast走两步
	if(slow == fast) break; // 判断相遇
}

if (fast == Null || fast->next == Null) return false;

fast = head; // fast从head出发
while(slow != fast) {
	slow = slow->next;
	fast = fast->next;
}

return slow;
```

最后 `slow` 和 `fast` 就是入口点
