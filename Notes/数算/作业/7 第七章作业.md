从图中删除一个处在 MST 中边，对于剩下的边，执行 Kruskal 算法得到一个树；对于 MST 中的所有边，重复这个过程，从得到的树中选出权值最小，且大于 MST 的权值的那个树

```
def Kruskal(graph):
	for edge in graph:
		minHeap.insert(edge)
	equalNum = graph.vertexNum
	MST = []
	while equalNum > 1 and not minHeap.empty():
		edge = minHeap.removeRoot()
		from, to = edge.fromVertex, edge.toVertex
		if find(from) != find(to):
			union(from, to)
			MST.append(edge)
			equalNum -= 1
	return MST

MST = Kruskal(graph)
res = NULL
minWeight = infinity
for edge in MST:
	graph.remove(edge)
	tmp = Kruskal(graph)
	if MST.totalWeight < res.totalWeight < minWeight:
		res = tmp
```

# 3

橙色代表加入集合，蓝色代表未加入集合，过程如下
- 给每个顶点整一个值，表示源点 s 目前到其的最短路径长度
- 维护一个子图，我们已知在这个子图内部 s 到其它节点的最短路径长度
- 选择值最小的节点加入子图，更新这个节点相连的节点的值
- 重复上一步直到所有节点都被加入子图

```
visited = [s]
for vertex in graph:
	d[vertex] = infinity
d[s] = 0
for edge in vertex.outEdges:
	d[edge.toVertex] = edge.weight

while visited.size() < graph.vertexNum:
	mind, minVertex = infinity, NULL
	for vertex in graph:
		if vertex not in visited and d[vertex] < mind:
			mind = d[vertex]
			minVertex = vertex
	visited.append(minVertex)
	for edge in minVertex.outEdges:
		d[edge.toVertex] = min(d[edge.toVertex], 
			d[minVertex] + edge.weight)
```
