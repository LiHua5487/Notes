
# 伪码

- 保持程序的主要结构
	- 类：`C, Pascal`
	- 赋值语句：`←`
	- 分支语句：`if ... then ... [else ...]`
	- 循环语句：`while, for, repeat ... until`
	- 转向语句：`goto`
	- 调用
	- 注释：`//...`
- 允许使用自然语言
- 常忽略数据结构、模块、异常处理等细节
- 常忽略变量说明

伪码示例（插入排序）

```
INSERTION-SORT(A)
1. for j ← 2 to length[A]
2.   do key ← A[j]  // 将 A[j] 插入排序的序列 A[1..j-1]
3.   i ← j - 1
4.   while i > 0 and A[i] > key
5.     do A[i+1] ← A[i]
6.     i ← i - 1
7.   A[i+1] ← key
```

>其中 `do` 表示 `for` `while` 的循环体的起始位置

# 复杂度
## 算法的复杂度

设 $f$ 和 $g$ 是定义域为自然数集 $N$ 上的函数，则
1. $f(n) = O(g(n))$ ：若存在正数 $c$ 和 $n_0$ 使得对一切 $n \geq n_0$ 有 $0 \leq f(n) \leq c g(n)$ 
2. $f(n) = \Omega(g(n))$ ：若存在正数 $c$ 和 $n_0$ 使得对一切 $n \geq n_0$ 有 $0 \leq c g(n) \leq f(n)$ 
3. $f(n) = o(g(n))$ ：对所有正数 $c > 0$ 存在 $n_0$ 使得对一切 $n \geq n_0$ 有 $0 \leq f(n) < c g(n)$ 
4. $f(n) = \omega(g(n))$ ：对所有正数 $c > 0$ 存在$n_0$使得对一切 $n \geq n_0$ 有 $0 \leq c g(n) < f(n)$ 
5. $f(n) = \Theta(g(n)) \iff f(n) = O(g(n))$ 且 $f(n) = \Omega(g(n))$ 
6. $O(1)$表示常数函数  

>小写版本像是 < ，而大写版本像是 ≤ （比如一个算法时间复杂度是 $o(n^2)$ ，意味着它比所有二次函数增长都慢，可能是 $n\log n$ 或更优）

>由 Stirling 公式可得，$n!=o(n^n),n!=\Omega(2^n)$ 

**定理**
1. 对于 $f(n)$ 和 $g(n)$ 是定义域为自然数集 $N$ 上的函数
	- 如果 $\lim_{n \to \infty} \frac{f(n)}{g(n)}$ 存在，并且等于某个常数 $c > 0$，那么
$$f(n) = \Theta(g(n)).$$
	- 如果 $\lim_{n \to \infty} \frac{f(n)}{g(n)} = 0$ ，那么
$$f(n) = o(g(n)).$$
	- 如果 $\lim_{n \to \infty} \frac{f(n)}{g(n)} = +\infty$ ，那么
$$f(n) = \omega(g(n)).$$
2. $O,\Omega,\Theta$ 具有传递性：$f=O(g),g=O(h) \Rightarrow f=O(h)$ 
3. $f=O(h),g=O(h) \Rightarrow f+g=O(h)$ 
	- 推论：$g=O(f) \Rightarrow f+g=\Theta(f)$ 

---

例子：下面的算法输入为大于 $2$ 的奇整数 $n$ ，输出为 true 或者 false 

```
1. s ← √n
2. for j ← 2 to s
3.   if j 整除 n
4.     then return false
5. return true
```

假设计算 $\sqrt{n}$ 可以在 $O(1)$ 时间完成，则该算法是 $O(\sqrt{n})$ ，但不是 $\Theta(\sqrt{n})$ ，因为如果取 $n$ 使得 $\sqrt n$ 为 3 的倍数，那这个算法运行时间是常数级别的（$j$ 取到 3 就完事了），不能以 $\Omega(\sqrt n)$ 为下界

## 问题的复杂度

**多项式时间的算法**：存在多项式 $p(n)$ ，使得算法的时间复杂度为 $O(p(n))$ 
**非多项式时间的算法**：不存在多项式 $p(n)$ ，可能包含指数项甚至更高阶
**可解的问题 / 难解的问题**：存在 / 不存在多项式时间的算法
**实际可计算的问题**：多项式时间可解的问题
