# 迭代法

## 直接迭代

例（汉诺塔）：汉诺塔的求解时间的递推方程为 $T(n)=2T(n-1)+1$ ，直接迭代法即可求解（直接代入）

$$
\begin{aligned}
T(n) &= 2T(n-1) + 1 \\
&= 2[2T(n-2) + 1] + 1 = 2^2T(n-2) + 2 + 1 \\
&= 2^3T(n-3) + 2^2 + 2 + 1 = \dots \\
&= 2^{n-1}T(1) + 2^{n-2} + \dots + 2 + 1 \\
&= 2^n - 1
\end{aligned}
$$

---

例（插入排序）：求插入排序的最坏时间复杂度

```
InsertSort(A, n) // A为n个数的数组
1. for j ← 2 to n  
2.    x ← A[j]  
3.    i ← j - 1    // 行3到行7把A[j]插入A[1..j-1]之中  
4.    while i > 0 and x < A[i] do  
5.        A[i+1] ← A[i]  
6.        i ← i - 1  
7.    A[i+1] ← x  
```

最坏情况下，当数组长度从 $n-1$ 增加到 $n$ ，需要额外处理最后一个元素，可得递推公式如下

$$
\begin{aligned}
W(n) &= W(n-1) + n - 1 \\
&= [W(n-2) + n - 2] + n - 1 = W(n-2) + (n-2) + (n-1) \\
&= [W(n-3) + n - 3] + (n-2) + (n-1) = \dots \\
&= W(1) + 1 + 2 + \dots + (n-2) + (n-1) \\
&= 1 + 2 + \dots + (n-2) + (n-1) = n(n-1)/2
\end{aligned}
$$

## 换元迭代

例（二分归并排序）：求二分归并排序的最坏时间复杂度

```
MergeSort(A, p, r) // 归并排序数组A[p..r]  
1. if p < r  
2.    then q ← ⌊(p+r)/2⌋  
3.    MergeSort(A, p, q)  
4.    MergeSort(A, q+1, r)  
5.    Merge(A, p, q, r)

Merge(A, p, q, r) // 将排序数组A[p..q]与A[q+1..r]合并  
1. x ← q - p + 1, y ← r - q    // x, y分别为两个子数组的元素数  
2. 将A[p..q]复制到B[1..x], 将A[q+1..r]复制到C[1..y]  
3. i ← 1, j ← 1, k ← p  
4. While i ≤ x and j ≤ y do  
5.    if B[i] ≤ C[j]    // B的首元素不大于C的首元素  
6.        A[k] ← B[i]    // 将B的首元素放到A中  
7.        i ← i + 1  
8.    else  
9.        A[k] ← C[j]  
10.       j ← j + 1  
11.   k ← k + 1  
12. if i > x then 将C[j..y]复制到A[k..r]    // B已经是空数组  
13. else 将B[i..x]复制到A[k..r]    // C已经是空数组
```  

归并排序一个长度为 $n$ 的数组，相当于先排序其左右两半，最后再合并起来，可得递推公式如下（只考虑比较次数，合并最坏耗时为 $n-1$ ）

$$
\begin{aligned}
\begin{cases}
W(n) = 2W(n/2) + n - 1 \\
W(1) = 0
\end{cases}
\end{aligned}
$$

这里涉及到 $n/2$ ，可以考虑换元方便计算，令 $n=2^k$ ，则

$$
\begin{aligned}
W(n) &= 2W(2^{k-1}) + 2^k - 1 \\
&= 2[2W(2^{k-2}) + 2^{k-1} - 1] + 2^k - 1 \\
&= 2^2W(2^{k-2}) + 2^k - 2 + 2^k - 1 \\
&= 2^2[2W(2^{k-3}) + 2^{k-2} - 1] + 2^k - 2 + 2^k - 1 \\
&= \dots \\
&= 2^kW(1) + k2^k - (2^{k-1} + 2^{k-2} + \dots + 2 + 1) \\
&= k2^k - 2^k + 1 \\
&= n \log n - n + 1
\end{aligned}
$$

但是实际上，由于  $n$ 实际上不一定 $2^k$ ，严谨起见可以用数学归纳法验证一下结果是不是对所有 $n$ 成立 

## 差消迭代

例（快速排序）：求快速排序的平均时间复杂度

先回顾一下快排过程：先确定一个轴值，把比它小的都放到左边，比它大的都放到右边，然后对左右两部分递归

确定轴值并排序的过程耗时 $O(n)$ ，对左右两部分递归，左右两部分的长度可能为 $(0,n-1),(1,n-2),\cdots,(n-1,0)$ ，每种情况出现概率为 $1/n$ ，可得递推公式如下

$$
\begin{cases}
T(n) = \frac{2}{n} \sum_{i=1}^{n-1} T(i) + O(n), \quad n \geq 2 \\
T(1) = 0
\end{cases}
$$

由于递推式存在求和式，很难一次性全代进去，可以考虑做差来处理这个求和项

$$
\begin{aligned}
nT(n) &= 2\sum_{i=1}^{n-1} T(i) + cn^2, \quad \text{c为某个常数} \tag{1} \\
(n-1)T(n-1) &= 2\sum_{i=1}^{n-2} T(i) + c(n-1)^2 \tag{2} \\
nT(n) &= (n+1)T(n-1) + O(n) \tag*{(1)-(2)}
\end{aligned}
$$
$$
\begin{aligned}
\frac{T(n)}{n+1} &= \frac{T(n-1)}{n} + \frac{c_1}{n+1} = \dots = c_1\left[\frac{1}{n+1} + \frac{1}{n} + \dots + \frac{1}{3}\right] + \frac{T(1)}{2} \\
&= c_1\left[\frac{1}{n+1} + \frac{1}{n} + \dots + \frac{1}{3}\right] = \Theta(\log n), \quad T(n) = \Theta(n \log n)
\end{aligned}
$$

递归树：把迭代展开的过程表示成树的形式

比如之前的 $W(n) = 2W(n/2) + n - 1$ ，它的递归树如下

---

假设有个递推公式如下，可得其递归树

$$
T(n) = T(n/3) + T(2n/3) + n
$$

![[算分/img/img2/image-1.png]]

最后会一直拆分到 1 ，由于右边分解更慢，层数会更多，可得总层数 $k$ 为

$$
\text{层数 } k: \quad n(2/3)^k = 1 \implies (3/2)^k = n \implies k = O(\log_{3/2} n)
$$

每层总和都是 $n$ （右边更慢分解到 1 ，后面几层的每层总和就是 $O(n)$ ），故总的时间复杂度为

$$
T(n) = O(n \log n)
$$

# 尝试法

得到递推式后，分别尝试 $T=c,T=cn,T=cn^2,T=n\log n$ 等结果，比如对于之前的快速排序，可以有以下猜测

$$
\begin{aligned}
(1) \ T(n) &= C\text{为常函数，左边}=O(1) \\
\text{右边} &= \frac{2}{n}C(n-1) + O(n) \\
&= 2C - \frac{2C}{n} + O(n) \\[1em]
(2) \ T(n) &= cn, \text{左边}=cn \\
\text{右边} &= \frac{2}{n} \sum_{i=1}^{n-1} ci + O(n) \\
&= \frac{2c}{n} \frac{(1+n-1)(n-1)}{2} + O(n) \\
&= cn - c + O(n) \\[1em]
(3) \ T(n) &= cn^2, \text{左边}=cn^2 \\
\text{右边} &= \frac{2}{n} \sum_{i=1}^{n-1} ci^2 + O(n) \\
&= \frac{2}{n} \left[\frac{cn^3}{3} + O(n^2)\right] + O(n) \\
&= \frac{2c}{3}n^2 + O(n) \\[1em]
(4) \ T(n) &= cn\log n, \text{左边}=cn\log n \\
\text{右边} &= \frac{2c}{n} \sum_{i=1}^{n-1} i\log i + O(n) \\
&= \frac{2c}{n} \left[\frac{n^2}{2}\log n - \frac{n^2}{4\ln 2} + O(1)\right] + O(n) \\
&= cn\log n + O(n)
\end{aligned}
$$

注意 $(2),(3)$ 中虽然量级相同，但是前面几项和假设对不上（$cn-n,\frac{2c}{3}n^2$），而 $(4)$ 中能对的上，而且后面 $O(n)$ 量级较小可忽略

此外，在 $(4)$ 中使用了积分结果作为求和的近似

$$
\begin{aligned}
\int_2^n x \log x dx &= \int_2^n \frac{x}{\ln 2} \ln x dx \\
&= \frac{1}{\ln 2} \left[\frac{x^2}{2} \ln x - \frac{x^2}{4}\right]_2^n \\
&= \frac{1}{\ln 2} \left(\frac{n^2}{2} \ln n - \frac{n^2}{4}\right) - \frac{1}{\ln 2} \left(\frac{4}{2} \ln 2 - \frac{4}{4}\right) \\[1em]
\sum_{i=1}^{n-1} i \log i &= \frac{n^2}{2} \log n - \frac{n^2}{4 \ln 2} + O(1)
\end{aligned}
$$

# 主定理（Master 定理）

设 $a \geq 1, b > 1$ 为常数，$f(n)$ 为函数，$T(n)$ 为非负整数，且 

$$
T(n) = aT(n/b) + f(n)
$$

则有以下结论
1. 若 $f(n) = O(n^{\log_b a - \varepsilon}),\varepsilon > 0$ ，那么 $T(n) = \Theta(n^{\log_b a})$ 
2. 若 $f(n) = \Theta(n^{\log_b a})$ ，那么 $T(n) = \Theta(n^{\log_b a} \log n)$ 
3. 若 $f(n) = \Omega(n^{\log_b a + \varepsilon}),\varepsilon > 0$ ，且对于某个常数 $c < 1$ 和充分大的 $n$ 有 $a f(n/b) \leq c f(n)$ ，那么 $T(n) = \Theta(f(n))$ 

>这里 $n^{\log_b a}$ 来源于 $aT(n/b)$ 的迭代展开，而三种分类情况实际上就是在比较 $f(n)$ 和它的量级关系

---

证明：设 $n = b^k$ ，换元迭代可得

$$
\begin{aligned}
T(n) &= aT\left(\frac{n}{b}\right) + f(n) \\
&= a\left[aT\left(\frac{n}{b^2}\right) + f\left(\frac{n}{b}\right)\right] + f(n) \\
&= a^2T\left(\frac{n}{b^2}\right) + af\left(\frac{n}{b}\right) + f(n) \\
&= \ldots \\
&= a^kT\left(\frac{n}{b^k}\right) + a^{k-1}f\left(\frac{n}{b^{k-1}}\right) + \ldots + af\left(\frac{n}{b}\right) + f(n) \\
&= a^kT(1) + \sum_{j=0}^{k-1} a^j f\left(\frac{n}{b^j}\right) \\
&= c_1n^{\log_ba} + \sum_{j=0}^{k-1} a^j f\left(\frac{n}{b^j}\right), \quad T(1) = c_1
\end{aligned}
$$

以第一种情况 $f(n) = O(n^{\log_b a - \varepsilon}),\varepsilon > 0$ 为例，可得

$$
\begin{aligned}
T(n) &= c_1n^{\log_b a} + \sum_{j=0}^{k-1} a^j f\left(\frac{n}{b^j}\right) \\
&= c_1n^{\log_b a} + O\left(\sum_{j=0}^{\log_b n-1} a^j \left(\frac{n}{b^j}\right)^{\log_b a - \epsilon}\right) \\
&= c_1n^{\log_b a} + O\left(n^{\log_b a - \epsilon} \sum_{j=0}^{\log_b n-1} \frac{a^j}{(b^{\log_b a - \epsilon})^j}\right) \\
&= c_1n^{\log_b a} + O\left(n^{\log_b a - \epsilon} \sum_{j=0}^{\log_b n-1} (b^\epsilon)^j\right) \\
&= c_1n^{\log_b a} + O\left(n^{\log_b a - \epsilon} \frac{b^{\epsilon \log_b n} - 1}{b^\epsilon - 1}\right) \\
&= c_1n^{\log_b a} + O\left(n^{\log_b a - \epsilon} n^\epsilon\right) = O\left(n^{\log_b a}\right)
\end{aligned}
$$

第二种情况类似，对于第三种情况，利用条件中的不等式进行放缩

$$
\begin{aligned}
T(n) &= c_1n^{\log_b a} + \sum_{j=0}^{k-1} a^j f\left(\frac{n}{b^j}\right) \\
&\leq c_1n^{\log_b a} + \sum_{j=0}^{\log_b n - 1} c^j f(n) \quad (af\left(\frac{n}{b}\right) \leq cf(n)) \\
&= c_1n^{\log_b a} + f(n) \frac{c^{\log_b n} - 1}{c - 1} \\
&= c_1n^{\log_b a} + \Theta(f(n)) \quad (c < 1) \\
&= \Theta(f(n))
\end{aligned}
$$

---

不能使用主定理的情况
- 形式不符合
- 形式符合，但是不满足分类条件

比如 $T(n) = 2T(n/2) + n \log n$ 不能使用主定理，只能使用递归树，结果等于  $T(n)=n\log^2 n$ ，因为
- 不存在 $c<1$ 使得 $2(n/2)\log(n/2)\leq cn\log n$  
- 不存在 $\varepsilon>0$ 使得 $f(n)=n\log n=\Omega(n^{1+\varepsilon})$ （因为 $n^\epsilon$ 增长比 $\log n$ 快）

# 取整符号

先把取整符号去掉，得到一个解，再代回去用数学归纳法验证

例：求 $T(n) = 2T(\lfloor n/2 \rfloor) + n$ 

根据 $T(n) = 2T(n/2) + n$  ，可得 $T(n) = O(n \log n)$ ，猜想原递推方程的解的阶是 $O(n \log n)$ ，即需证明 $T(n) \leq cn \log n$，用数学归纳法（ $n=1$ 情况单独讨论）
- 归纳基础：  考虑 $T(2)$ ，$T(2) = 2T(\lfloor 1 \rfloor) + 2 = 4 \leq 2 \times 2 \log 2, \, c = 2$  
- 归纳步骤：  假设对于小于 $n$ 的正整数命题为真，那么  

$$
\begin{aligned}
T(n) &= 2T\left(\left\lfloor \frac{n}{2} \right\rfloor\right) + n \\
&\leq 2c \left\lfloor \frac{n}{2} \right\rfloor \log\left(\left\lfloor \frac{n}{2} \right\rfloor\right) + n \\
&\leq 2c \frac{n}{2} (\log n - \log 2) + n \\
&= c n \log n - c n + n \\
&\leq c n \log n, \quad c = 2
\end{aligned}
$$
